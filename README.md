# Tugas 5 - Notes Navigation App

Aplikasi Notes dengan fitur navigasi antar layar menggunakan Compose Multiplatform.

## Fitur Utama
- Bottom Navigation: Notes, Favorites, Profile
- Notes -> Note Detail dengan passing `noteId`
- Floating Action Button untuk Add Note
- Edit Note dengan passing `noteId`
- Back navigation dari semua screen
- Tab Profile tetap memakai tampilan Tugas 4

## Struktur Folder
- `navigation/`
- `screens/`
- `components/`

## Catatan
- Profile page dipertahankan dari Tugas 4
- Notes dan Favorites dimodifikasi agar lebih modern dan tidak boring
